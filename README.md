# Recipe Generator App
The Recipe Generator App is a powerful tool that brings creativity and convenience to your kitchen. With this app, you can say goodbye to the dilemma of deciding what to cook based on the ingredients you have on hand. Our advanced algorithm and extensive recipe database will suggest unique and personalized meal ideas tailored to your preferences.

## Features
1. **Ingredient-Based Recipe Generation:** Simply input the ingredients you have in your pantry, and the app will generate a list of recipes that can be prepared using those ingredients. No more wasted food or last-minute trips to the grocery store!

2. **Preferred Cuisine Selection:** Customize your recipe suggestions by selecting your preferred cuisines. Whether you're craving Italian, Asian, Mexican, or any other type of cuisine, the app will provide recipes that match your taste.

3. **Save Your Favorite Recipes:** Found a recipe that you absolutely love? Save it to your favourites list for quick access later. Never lose track of your most cherished culinary discoveries.

4. **Create Grocery Lists:** Planning your next grocery shopping trip is now a breeze. The app allows you to generate a comprehensive grocery list based on the recipes you've selected, ensuring that you have all the necessary ingredients at hand.

## Technologies Used
The Recipe Generator App has been developed using the Flutter framework. Flutter is a cross-platform development framework that allows us to build beautiful, responsive, and fast apps for both Android and iOS. By leveraging the power of Flutter, we were able to create a seamless and consistent user experience across multiple platforms.

## Installation
To install and run the Recipe Generator App on your device, follow these steps:
1. Clone this repository to your local machine using ***git clone https://github.com/US2002/RecipeApp.git***.
2. Navigate to the project directory.
3. Ensure that you have Flutter installed on your system. If not, refer to the official Flutter documentation for installation instructions.
4. Connect your device or emulator.
5. Run the command flutter run to start the app.
